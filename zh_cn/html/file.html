  目录:

	bin/     		函数文件: bin/include 及其他功能文件存放目录
	conf/    		配置文件目录, 唯一配置文件: conf/eminfo.ini
	doc/     		文档和自述文件存放目录
	handler/ 		故障响应程序的存放目录, 其中有eminfo的三个内置handler: postlog, sendmail, takesnap
	log/     		日志文件目录, log/backup/是日志回滚目录, log/snapshot/是系统快照目录
	plugin/  		插件程序存放目录
	tmp/     		临时文件存放目录
	opt/     		其他

  
  文件:

	bin/
	  include		函数文件: 包含eminfo的全局变量定义和公共函数定义
	  funlib		函数文件: 其他相关函数
	  conv2py		负责汉字转换拼音的脚本 
	  precheck		检查当前系统环境能否运行eminfo

	handler/
	  sendmail		内置handler,负责发送邮件的脚本
	  takesnap		内置handler,负责系统快照的脚本
	  postlog		内置handler,负责上报信息的脚本

	conf/
	  eminfo.ini		唯一的配置文件

	log/
	  eminfo.log		主日志: 记录eminfo运行日志和用户插件运行日志
	  inner.log		内部功能插件运行日志
	  postlog.log		信息上报日志
	  sendmail.log		发送信件日志
	  takesnap.log		系统快照日志

	opt/
	  ...			某些插件所需的资源文件和样例文件

	plugin/
	  ...			各个插件
